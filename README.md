Sistem para calcular o preço de serviços de usinagem em mdf.

Considerando uma constante de R$ 128,57 para cada 1m², o sistema calcula o preço final do serviço a ser cobrado, levando em conta as medidas já definidas. Essa constante leva em conta a média de preço atual do MDF de 3mm.

Para definir as medidas, basta selecionar a opção 1 e definir sempre em milímetros.

A opção 2 irá gerar o preço baseado nas medidas definidas.

Futuramente será adicionado cálculo para MDF de 6mm e acrílico.
